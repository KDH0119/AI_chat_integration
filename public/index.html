API Key 발급 가이드

1. Google Gemini API Key

  1. AI STUDIO 접속: https://aistudio.google.com/api-keys
  2. 왼쪽 하단 Get API Key 클릭
  3. API 키 만들기 클릭
  4. 프로젝트 선택 → Create project → 이름 입력 후 생성
  5. 키 이름 지정 후 생성
  6. 결제 설정 클릭
  7. 결제 정보, 카드 정보 등 입력 후 가입
  8. 생성된 API KEY를 ai_chat_main 사이트 입력란에 붙여넣기
  9. 완료

2. NovelAI API Key

  1. NovelAI 사이트 접속: https://aistudio.google.com/api-keys
  2. Login
  3. User Setting → Account 클릭
  4. Get Persistent API Token 클릭
  5. 발급받은 API Key 복사
  6. 원하는 입력란에 붙여넣고 사용
  7. 완료
